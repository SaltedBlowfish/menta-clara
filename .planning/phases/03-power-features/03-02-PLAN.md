1. **search-notes.ts** -- Create `searchNotes` async function:
   ```typescript
   interface SearchResult {
     noteId: string;
     noteType: 'daily' | 'permanent' | 'weekly';
     snippet: string;
     title: string;
   }
   export async function searchNotes(query: string, workspaceId: string): Promise<ReadonlyArray<SearchResult>>
   ```
   Implementation:
   - Open database, get all keys from NOTES_STORE
   - For `workspaceId === 'personal'`, scan keys matching `daily:*`, `weekly:*`, `permanent:*` (no ws: prefix)
   - For other workspaces, scan keys matching `ws:${workspaceId}:daily:*`, etc.
   - For each note, extract plain text from TipTap JSONContent by recursively walking `content` arrays and collecting `text` properties from text nodes
   - Case-insensitive search: `plainText.toLowerCase().includes(query.toLowerCase())`
   - Return matches with `title` derived from note ID:
     - `daily:YYYY-MM-DD` -> format with date-fns `format(parseISO(dateStr), 'MMMM d, yyyy')`
     - `weekly:YYYY-Wnn` -> `Week nn`
     - `permanent:uuid` -> look up name from `setting:permanentNames` record in DB
   - `snippet`: 80 chars of text centered around the first match position
   - Limit results to 20
   - Create helper `extractText(json: JSONContent): string` that walks the TipTap JSON tree

2. **highlight-snippet.ts** -- Create `highlightSnippet` function:
   ```typescript
   interface HighlightedPart {
     highlight: boolean;
     text: string;
   }
   export function highlightSnippet(snippet: string, query: string): ReadonlyArray<HighlightedPart>
   ```
   Split snippet into parts where matches are marked with `highlight: true`. Case-insensitive splitting.

   - Tracks `isOpen`, `inputValue`, `activeIndex`
   - `isCommandMode = inputValue.startsWith('>')`
