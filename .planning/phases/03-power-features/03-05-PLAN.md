1. **tiptap-markdown.ts** -- TipTap JSON to/from markdown conversion:
   ```typescript
   import type { JSONContent } from '@tiptap/react';
   ```
   - `jsonToMarkdown(json: JSONContent): string` -- Use @tiptap/markdown's serializer. Create a headless TipTap editor instance with the same extensions as the app (import `editorExtensions`), set content to json, call the markdown serialization API.
   - `markdownToJson(markdown: string): JSONContent` -- Use @tiptap/markdown's parser. Create a headless editor, parse markdown, return getJSON().
   - These are pure conversion functions, no React hooks. Use TipTap's headless editor (no DOM needed).
   - Handle edge case: if @tiptap/markdown doesn't handle image nodes with `data-image-id` attribute, manually post-process the markdown to insert image references.

2. **export-workspace.ts** -- Export function:
   ```typescript
   import { strToU8, zipSync } from 'fflate';

   export async function exportWorkspace(workspaceId: string): Promise<Blob>
   ```
   Steps:
   - Get all notes from NOTES_STORE matching the workspace prefix (or unprefixed for 'personal')
   - For each note, convert TipTap JSON content to markdown via `jsonToMarkdown`
   - Organize into folders by note type:
     - `daily/YYYY-MM-DD.md` (strip `daily:` prefix)
     - `weekly/YYYY-Wnn.md` (strip `weekly:` prefix)
     - `permanent/{name}.md` (look up name from `setting:permanentNames`)
   - Get all images from IMAGES_STORE matching workspace prefix
   - For each image, convert blob to Uint8Array, add to `images/{imageId}.{ext}` based on mimeType
   - Rewrite image src in markdown from blob URLs to relative paths `../images/{filename}`
   - Include templates as `templates/{name}.md`
   - Include workspace settings as `settings.json` (template defaults, permanent note names)
   - Create zip with `zipSync` from fflate, return as Blob
   - Trigger download: `const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = '{workspaceName}-export.zip'; a.click(); URL.revokeObjectURL(url);`

3. **import-workspace.ts** -- Import function:
   ```typescript
   export async function importWorkspace(file: File, workspaceId: string): Promise<number>
   ```
   Steps:
   - If file is `.zip`: extract with `unzipSync` from fflate
   - If file is `.md`: treat as single note import
   - For each markdown file in the zip:
     - Determine note type from folder path (daily/, weekly/, permanent/)
     - Convert markdown to TipTap JSON via `markdownToJson`
     - Build note ID from filename + workspace prefix
     - Check if note already exists in IndexedDB
     - If exists: append imported content below existing content with `## Imported on {format(new Date(), 'MMMM d, yyyy')}` heading (as TipTap JSONContent heading node + imported content nodes)
     - If not exists: create new note
   - For images in images/ folder: store each as blob in IMAGES_STORE
   - For templates in templates/ folder: save as templates
   - For settings.json: merge template defaults (don't overwrite existing settings, only add missing)
   - Return count of imported notes
   - Error handling: wrap in try/catch, distinguish between invalid file format and parse errors per UI-SPEC error copy
