1. Create `src/shared/use-persisted-state.ts`:
   - Generic hook: `usePersistedState<T>(key: string, defaultValue: T): [T, (value: T) => void]`
   - Uses `getDatabase()` from `../storage/database` to read/write from NOTES_STORE
   - Key format: the `key` param directly (callers pass `setting:theme`, `setting:paneRatio`, etc.)
   - On mount: read from IndexedDB, parse, set state. While loading, return defaultValue.
   - On set: update React state AND write to IndexedDB (no debounce needed for settings).
   - Store format in IndexedDB: `{ id: key, value: T, updatedAt: number }`

3. Create `src/theme/use-theme.ts`:
   - Export `useTheme(): { theme: 'light' | 'dark'; toggle: () => void }`
   - Uses `usePersistedState('setting:theme', null)` where null means "use system"
   - On mount and on change: set `document.documentElement.setAttribute('data-theme', resolvedTheme)`
   - Resolved theme: if persisted value is 'light' or 'dark', use it. If null, check `window.matchMedia('(prefers-color-scheme: dark)').matches` to determine current system preference.
   - `toggle()`: if current resolved theme is 'light', persist 'dark' and vice versa.
   - Listen to `matchMedia` change event to update if user has no explicit preference and system changes.

   - Uses internal `useState` for expanded (initialized from `defaultExpanded`, default `true`)
   - Renders:
     ```tsx

           className="collapsible-section-toggle"
           onClick={handleToggle}
           type="button"
         >
           <svg className={`collapsible-chevron${expanded ? ' collapsible-chevron-expanded' : ''}`} ...>

       </h3>

         className="collapsible-section-content"
         hidden={!expanded}
         id={`${sectionId}-content`}
       >

       </div>
     </section>
     ```
