2. Create `src/layout/use-pane-ratio.ts`:
   - Export `usePaneRatio()` returning `{ containerRef: RefObject<HTMLDivElement | null>; onPointerDown: (e: PointerEvent) => void; ratio: number }`
   - Uses `usePersistedState('setting:paneRatio', 60)` for the ratio (percentage for left pane)
   - `onPointerDown` handler:
     - Calls `e.currentTarget.setPointerCapture(e.pointerId)` (the currentTarget is the divider)
     - Sets a `dragging` ref to true
   - Attach `pointermove` and `pointerup` listeners to the divider element via the returned `onPointerDown`
   - On `pointermove`: calculate `newRatio = (e.clientX / containerRef.current.getBoundingClientRect().width) * 100`
   - Clamp: compute min/max from 250px min pane width: `minRatio = (250 / containerWidth) * 100`, `maxRatio = 100 - minRatio`
   - On `pointerup`: `releasePointerCapture`, set dragging to false, persist ratio via setRatio
   - NOTE: Use `onPointerMove` and `onPointerUp` props on the divider element rather than document-level listeners since setPointerCapture routes all events to the captured element

3. Create `src/layout/pane-divider.tsx`:
   - Export `PaneDivider` with props: `{ onPointerDown: (e: React.PointerEvent) => void; ratio: number }`

     ```tsx
       aria-controls="daily-pane"
       aria-label="Resize panes"
       aria-orientation="vertical"
       aria-valuemax={100}
       aria-valuemin={0}
       aria-valuenow={Math.round(ratio)}
       className="pane-divider"
       onPointerDown={onPointerDown}
       onPointerMove={...}
       onPointerUp={...}
       role="separator"
       tabIndex={0}
     />
     ```
   - NOTE: The pointermove/pointerup handlers need to come from the parent hook. Adjust the props interface to accept `onPointerMove` and `onPointerUp` as well, OR have the hook return all three handlers.
   - Keyboard support: onKeyDown handler for Left/Right arrow keys to adjust ratio by 2% increments
